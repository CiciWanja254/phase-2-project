# ExchangePal

This is a simple web application that displays and enables the users to vies currency exchange rates. Users can also and leave feedback.

## Description

This project seeks to create an app that allows users to easily exchange their currency and also compare current exchange rates with past exchange rates from the comfort of their home.

## Getting Started

### Executing program

#1) Clone this repository.

```
git clone git@github.com:CiciWanja254/phase-2-project.git

```

#2) Run `npm start` in the terminal to start the application.

The web page will load and display the ExchangePal application. The landing page would have a navigation bar that a user would click to view various aspects of the page.

Users can view the history, current exchange rates and finally have their comments in the final page and also react to other users comments by simply entering their names and comment in the appropriate fields and click the "Submit" button. Your comment will be added to the list of comments below the form

### Technologies

- HTML
- CSS
- JavaScript
- React

This project uses the Exchange Rates API to fetch the exchnage rates. You can get an API key by following the instructions on the NExchange Rates API website.https://exchangeratesapi.io/.

- Functionality
  React is used to fetch the exchnage rates from the Exchange Rates API and displays it on the page. The user can veiw the rates of the dolar against other currencies, the current rates and the history.The Reviews page has the comment section with a like button increments a counter each time it is clicked on every comment, which are added to the comment section below.

- Styling
  The page is styled using custom CSS. The font family used is Montserrat.
